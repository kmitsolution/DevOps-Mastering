##  Remote Repository

A **remote repository** is a **central location** where your Git code is stored so that **multiple developers can collaborate**.

<img width="721" height="419" alt="image" src="https://github.com/user-attachments/assets/3d82d7f5-2ddd-48cd-abcc-b119fa16010f" />



### Key points

* Hosted on a server/cloud
* Shared by teams
* Used for **push, pull, clone, pull requests**
* Acts as a **single source of truth**

### Examples of remote repositories

* GitHub
* GitLab
* Bitbucket
* Azure Repos
* AWS CodeCommit

**GitHub** is a **cloud-based Git repository hosting platform** that helps teams:

* Store code
* Collaborate using Git
* Review code using Pull Requests
* Automate CI/CD workflows

GitHub is the **most popular Git hosting platform** in the world.
